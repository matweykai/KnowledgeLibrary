Inverted Residual Block: 
	- Это противоположность Residual Block из ResNet в 
	  архитектуре

	- Тк кол-во параметров сильно уменьшается благодаря 
	  Depth-wise свёртке, то можно увеличить кол-во каналов

	- На этой идее и сделали Inverted Residual Block, где вместо
	  уменьшения кол-ва внутренних слоёв их увеличили

	- Т.е структура Residual Block примерно такая(не обязательно
	  делятся пополам, просто становится меньше):

		N каналов -> N/2 каналов -> N/2 каналов -> N каналов

	  И первый слой объединяется с последним с помощью сложения
	
	- А вот архитектура для Inverted Residual Block:
		Conv 1x1 -> Dwise 3x3 -> Conv 1x1 -> Sum
		N каналов   K > N кан    N каналов   N каналов

	- Т.е тут вместо уменьшения кол-ва каналов внутри блока их
	  увеличивают
