ROI pooling - специальный слой, который используется в Fast R-CNN,
	      для обработки изображений любого размера

> Алгоритм:
	1. На основании заранее заданного числа P(размера конечного
	   изображения) вычисляются параметры:
		W_s = ROI_Width / P
		H_s = ROI_Height / P
	
	2. ROI разбивается на непересекающиеся области размера 
	   W_s х W_h. Крайние области больше или меньше по размерам,
	   чтобы захватить ROI полностью
	
	3. Для каждой такой области и каждого её канала выполняется
	   MaxPooling
	
	4. Получаем изображение размера P x P, которое можно 
	   подавать на классификацию
