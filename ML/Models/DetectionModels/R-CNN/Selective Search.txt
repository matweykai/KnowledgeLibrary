* Данный алгоритм пытается найти приближения объектов на изображении,
  которые дальше используются для детекции

> Алгоритм:
	1. Генерируем огромное количество маленьких сегментов(автор 
	   делал это с помощью метода из статьи Efficient Graph-Based
	   Image Segmentation)

	2. Рекурсивно сливаем маленькие сегменты в большие(на 
	   основании различных мер близости)

	3. Используем данные сегменты, как приближения для детекции

* На основании же каких мер мы сливаем сегменты?

	-> Color Similarity (сравниваются гистограмы цветов)
	-> Texture Similarity (на основании производных)
	-> Size Similarity
	-> Fill Similarity

* Все они суммируются в одно значение, на основании которого и
  сливаются регионы
