Region Proposal Network - это сеть, которая проходит по картинке
		некоторым окном и применяет к нему box'ы 
		определённого размера, которые называются anchor

* Данная сеть использует anchor'ы с различными соотношениями сторон 
  и размерами

* RPN базируется на выходах основного свёрточного слоя, что ускоряет
  модель

* Для тренировки слоя генерируются как positive, так и negative 
  примеры

Алгоритм:

	1) На выходах свёрточного слоя выставляются якоря

	2) К каждому якорю по сути привязан anchor box определённого
	   размера

	3) К каждой ячейке якоря применяется 1х1 свёртка, которая
	   используется для определения объекта в некотором боксе и
	   смещения якорного(offset) бокса
	
	   * После свёртки 1х1 получаем: 
	     Наличие объекта(клас.): (B, C_in, W, H) -> (B, 2k, W, H)
	     Регрессия(offset): (B, C_in, W, H) -> (B, 4k, W, H)
	     где k - количество шаблонных боксов

	4) Находятся боксы с самыми высокими objectness score
	   и получаются координаты бокса в оригинальном изображении
	  
	5) Для каждого anchor бокса находится ground truth box,
	   который используется для тренировки RPN слоя
	   
	   * Для одного anchor box - один gt box!
	   * Из тех anchor box'ов, для которых не нашлось ни одного
	     gt box'a с IOU выше порога, получаются негативные 
	     примеры для RPN сети
