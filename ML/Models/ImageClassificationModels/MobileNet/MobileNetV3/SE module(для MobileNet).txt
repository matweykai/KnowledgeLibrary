* В архитектуру Inversed Residual Block теперь входит SE блок

* Данный блок позволяет взвешивать интересующие нас каналы, т.е
  реализует механизм поканального внимания

Архитектура нового Inversed Residual Block'a:
	1) Расширяем вход с помощью свёртки 1х1
	
	2) Выполняем Depth-wise свёртку
	
	3) Делаем AvgPooling для каждого канала
	
	4) Умножаем на FC с ReLU
	
	5) Умножаем на FC с hard sigmoid
	
	6) Умножаем на полученные веса выход со 2 шага

	7) Сжимаем выход с помощью 1х1 свёртки
	
	8) Складываем вход в данный блок с выходом 7)
