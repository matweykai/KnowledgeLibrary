Population Stability Index - это метод, который позволяет оценить 
		стабильность некоторого признака во времени

* PSI оценивает предсказания модели во время тренировки и теста, 
  чем выше показатель, тем более разные распределения признаков

* Когда PSI применяют к признакам модели, то это называют CSI
  Characteristic Stability Index

* С помощью данного коэффициента нельзя увидеть, что целевая 
  переменная изменилась, можно увидеть только дрифт самих признаков

> Алгоритм:

	1. Выборки использованные при тренировке и тесте сортируются
	   по возрастанию

	2. Потом они делятся по квантилям на n промежутков

	3. Для каждого промежутка вычисляем долю попавших в него
	   значений

	4. Вычисляем по формуле PSI:

		PSI = SUM[ (test_i - train_i) * log(test_i / train_i) ]

* Критерии оценки индекса:

	       PSI < 0.1: изменения модели не требуются
	0.2 > PSI >= 0.1: нужно немного изменить модель
	      PSI >= 0.2: требуются существенные изменения модели
			  (перекалибровка или перестроение)
