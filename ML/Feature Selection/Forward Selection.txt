Forward Feature Selection - жадный алгоритм отбора признаков, который
                            перебирает различные комбинации признаков

Алгоритм:
	1) У нас нет ни одного признака
	2) Проходим по выборке, берём признаки(пока один) и смотрим 
           на важность признака по p-value
	3) Из всех признаков отбираем тот, который имеет наименьшее
	   значение p-value
	4) Используя найденные признаки, добавляем по очереди новых
	   кандидатов
	5) Обучаем модель
	6) Вычисляем p-value для каждого кандидата
	7) Отбираем лучший признак себе
	8) Повторяем всё с пункта 4, пока p-value будет меньше 
	   некоторого порога

* Данный алгоритм не рекомендуется по ряду причин: p-value 
  неправильно используется и оценки важности получаются смещёнными
