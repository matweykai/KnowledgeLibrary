* Координаты пикселей изображения отсчитываются по левой системе 
  координат, т.е (0, 0) это верхний левый угол изображения, а оси 
  идут так:


  *-----X>
  |
  |
  |
  |
  |
  Y
  V

* При этом принято обозначать множество всех значений, которое может
  принимать пиксель как ꭥ:
 
	ꭥ = { (x, y): 0 <= x <= N_cols && 0 <= y <= N_rows }
